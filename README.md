# Taskify -- Organize Today, Achieve Tomorrow

## 📌 Project Description

**Taskify** is a simple and user‑friendly task management website built
using **HTML, CSS, and JavaScript**.\
It helps users organize their daily tasks efficiently by allowing them
to add, view, and manage tasks in an organized interface.

The website is designed with a clean and attractive UI that includes
emojis, motivational quotes, and easy navigation.\
Taskify encourages productivity by helping users plan their work and
achieve their goals effectively.

------------------------------------------------------------------------

## 🚀 Features

-   📱 **Responsive Design** -- Works on mobile, tablet, and desktop
    devices.
-   🧭 **Navigation Bar** -- Easy navigation between pages such as Home,
    Add Task, View Tasks, and About.
-   ➕ **Add Tasks** -- Users can create and store tasks.
-   📋 **View Tasks** -- Displays all saved tasks in an organized
    layout.
-   ✏ **Basic CRUD Operations** -- Create, Read, Update, and Delete
    tasks.
-   💡 **Motivational Quotes & Emojis** -- Makes the interface engaging
    and encouraging.
-   📧 **Contact Form** -- Allows users to send feedback or messages.

------------------------------------------------------------------------

## ⚙️ Technologies Used

-   **HTML5** -- Structure of the website\
-   **CSS3** -- Styling and responsive layout\
-   **JavaScript** -- Dynamic functionality and task management logic

## ▶️ How to Run the Project

1.  Open the project folder.
2.  Locate the file **index.html**.
3.  Double‑click the file or open it in a browser.
4.  The **Taskify website** will run locally in your browser.
